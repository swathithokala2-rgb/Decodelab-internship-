# Online Book Store

## Project Description

Online Book Store is a full-stack web application developed as part of the DecodeLabs Internship Program. The application allows users to browse books, search for books, manage profiles, place orders, and download invoices. It uses MongoDB Atlas for database management and Node.js with Express.js for backend services.

## Features

* User Signup
* User Login
* Forgot Password
* JWT Authentication
* Book Listing
* Search Books
* Add Book
* Update Book
* Delete Book
* Shopping Cart
* Order Management
* Invoice PDF Generation
* User Profile Management
* Address Management

## Project Structure

* project_1.html – Home Page
* project_1.css – Styling
* project_1.js – Frontend Logic
* project_2.js – Backend APIs
* signup.html / signup.js – User Registration
* login.html / login.js – User Login
* admin.html / admin.js – Book Management
* profile.html / profile.js – User Profile
* order.html / order.js – Order Management

## How to Run

### Backend

```bash
npm install
node project_2.js
```

### Frontend

```bash
python3 -m http.server 5500
```

http://localhost:5500/project_1.html
