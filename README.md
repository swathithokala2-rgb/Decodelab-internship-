# Online Book Store

## Project Description

Online Book Store is a full-stack web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

* User Signup
* User Login
* Forgot Password
* JWT Authentication
* Book Listing
* Search Books
* Shopping Cart
* Order Management
* Invoice PDF Download
* Admin Panel
* Add/Delete Books
* User Profile

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## How to Run

### Frontend

python3 -m http.server 5500

### Backend

node project_2.js
